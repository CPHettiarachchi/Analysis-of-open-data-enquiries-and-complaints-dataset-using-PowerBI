# Open Data Enquiries & Complaints Analysis (Power BI)

## Project Overview
This project presents a comprehensive data analysis of the **Open Data Enquiries and Complaints Dataset (2010–2014)**.  

The objective is to uncover patterns, trends, and insights related to complaint submissions, categories, statuses, and resolution outcomes using **SQL Server** and **Power BI**.

The analysis transforms raw complaint records into meaningful insights that support **data-driven decision-making** and improved organizational responsiveness.

---

## Key Objectives
- Analyze temporal trends in complaints over time  
- Identify major complaint categories and patterns  
- Evaluate complaint statuses (accepted, rejected, unresolved)  
- Examine resolution outcomes (conciliation, withdrawal, referral)  
- Design interactive dashboards for better insight exploration  

---

## Tools & Technologies
- **Power BI** – Data visualization & dashboard creation  
- **SQL Server** – Data querying and transformation  
- **Excel** – Data preprocessing  
- **DAX** – KPI calculations and analytics  

---

## Dataset Information
- 📅 Time Period: July 2010 – June 2014  
- 📁 Total Records: 7,000+  
- 📌 Key Fields:
  - Complaint Date  
  - Submission Method  
  - Complaint Type  
  - Complaint Status  
  - Category (e.g., Disability, Race, Harassment)  
  - Resolution Outcome  

---

## Key Insights
- Disability, Race, and Sexual Harassment are among the top complaint categories  
- A significant portion of complaints are accepted, but many remain unresolved  
- Common resolution outcomes include conciliation and withdrawal  
- Seasonal trends show fluctuations in complaint volume  
- Data anomalies highlight the importance of proper validation  

---

## Dashboard Features
- Summary KPIs (Total complaints, resolution rates)  
- Time-series analysis (monthly/yearly trends)  
- Category-wise complaint breakdown  
- Geospatial visualization (regional patterns)  
- Drill-through and interactive filtering
